# 2026-06-30 - Release 0.8.1 Session

## Goal

Move EMC Locus toward the active three-surface local-first target by delivering
a small, executable, tested vertical slice for first-class attached documents.

## Delivered

- Added an agent-owned attached-document registry in `projects.sqlite`.
- Persisted document metadata only; file bytes remain outside SQLite.
- Added document audit events and pending sync outbox entries.
- Added local API routes for document registration, listing, owner-filtered
  listing, detail reads, and audit-event reads.
- Added Python `LocalAgentClient` coverage for the document routes.
- Updated the static GUI shell and architecture docs to name the three product
  surfaces: Locus Metrology, Locus Lab Management, and Locus Test Station.
- Added ADR 0003 for the three-surface architecture and document registry.
- Bumped the software version to `0.8.1`.

## Known Limits

- No document upload/download.
- No object-storage integration.
- No central PostgreSQL.
- No final document permissions or merge workflow.
- Attached documents currently use the `project_records` outbox domain with
  entity type `attached_document`.
