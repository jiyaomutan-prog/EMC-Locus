Goal: deliver `0.10.0 - LAB CONSOLE Template Studio v1`.

- Added `apps/lab-console` as a strict TypeScript React/Vite application.

- Added API-only create, clone, validate, save, submit, approve, derive,
  revision-history, audit, and system-status flows.
