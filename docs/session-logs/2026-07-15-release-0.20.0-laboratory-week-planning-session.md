# 2026-07-15 - Release 0.20.0 Laboratory Week Planning Session

## Objectif

Livrer une vue hebdomadaire exploitable du laboratoire : relire les créneaux de
plusieurs dossiers, les filtrer par ressource, déplacer un créneau prévu ou
confirmé avec concurrence optimiste, et conserver audit et outbox derrière le
Local Agent Rust.

## Audit initial

- La verticale 0.19 possède déjà un créneau typé, révisionné et rattaché à un
  dossier, avec conflits d'opérateur ou de lieu, transitions, audit et outbox.
- La lecture est limitée à un dossier ; il n'existe aucune vue hebdomadaire
  multi-dossiers dans l'agent ou LAB CONSOLE.
- La table actuelle contient les bornes, ressources, état et révision requis
  pour cette verticale. Aucune duplication de planning n'est nécessaire.
- Le dépôt ne propose pas d'action contrôlée pour déplacer un créneau. Une
  mise à jour directe contournerait les invariants, l'audit et l'outbox.
- Les index existants couvrent déjà le statut, le temps, l'opérateur et le lieu.
  La nécessité d'une migration supplémentaire doit être justifiée par les
  requêtes finales, pas supposée.
