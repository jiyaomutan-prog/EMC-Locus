# EMC Locus 0.6.7 - Historical Migration and HTTP E2E

## Delivered

Version `0.6.7` adds the migration and E2E confidence layer needed before the
`0.7.0` metrology vertical slice:

1. migration `0007_legacy_calibration_events.sql` backfills legacy
   `calibration_records` into `calibration_events`;
2. the backfill preserves original rows, certificate references, uncertainty
   JSON, file references, valid SHA-256 values, and legacy revision evidence;
3. migration coverage verifies reserved availability conversion, legacy
   calibration-event creation, and idempotent migration re-runs;
4. a real loopback HTTP server E2E test covers instrument registration,
   readiness refusal without calibration, calibration with certificate
   manifest, due-soon warning, out-of-service blocking, return-to-service,
   idempotent replay, replay conflict, restart persistence, audit reads, and
   sync outbox reads.
