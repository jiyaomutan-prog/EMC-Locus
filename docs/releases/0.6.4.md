# EMC Locus 0.6.4 - Calibration Events and Computed Status

## Delivered

Version `0.6.4` adds the first agent-backed calibration-event slice:

1. migration `0005_calibration_events_and_status.sql` adds
   `calibration_events` and per-instrument `calibration_due_warning_days`;
2. the Rust agent can record and list calibration events;
3. calibration events include controlled decisions, as-found/as-left status,
   uncertainty summary JSON, traceability reference, adjustment flag, operator,
   revision, and optional certificate document manifest;
4. the loopback API exposes calibration-event write/read routes and a computed
   calibration-status route;
5. computed status is derived from the latest calibration event, its decision,
   the due date, the requested check date, and the warning threshold.
