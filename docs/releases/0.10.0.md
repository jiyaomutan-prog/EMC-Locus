# EMC Locus 0.10.0 - LAB CONSOLE Template Studio v1

Version `0.10.0` replaces the temporary LAB web prototype with a real
React/TypeScript/Vite LAB CONSOLE served by the Rust local agent under `/lab/`.

## Delivered

- Template library loaded only from `/api/v1/test-templates`.
- Create and clone flows for revisioned test templates.
- Structured Template Studio sections for general metadata, variables, lock
  policy, instrumentation, sequence, limits, post-processing, revisions, audit,
  and read-only canonical JSON preview.
- Server-side validation through
  `POST /api/v1/test-template-definitions/validate`.
- Checksum-based draft save with visible CAS conflict handling.
- Submit, approve, and derive workflow using the existing revision aggregate.
- System status view for local agent/storage/schema state.
- Agent static serving for `/`, `/lab/`, `/lab/assets/*`, and SPA fallback.
- Versioned `apps/lab-console/dist` build for launch without Node.
- LAB launchers, full demo launcher, build launcher, and API-only demo seeding.
- Qt static mode decoupled from LAB web files via
  `apps/qt-console/demo/bootstrap.json`.
- Unit, backend, launcher, and Playwright E2E coverage for the delivered slice.

## Not Delivered

This release does not implement campaign instantiation, execution package
freezing, real acquisition, instrument drivers, SCPI/serial/CAN control, FFT,
reporting, authentication, RBAC, personnel competence checks, central
synchronization, or a full Qt rewrite.

## Operator Launch

```powershell
.\scripts\start-lab.ps1
.\scripts\start-lab.ps1 -SeedDemo
.\scripts\start-full-demo.ps1
.\scripts\stop-all.ps1
```

## Developer Build

```powershell
.\scripts\build-lab.ps1
.\scripts\start-lab.ps1 -Rebuild
```
