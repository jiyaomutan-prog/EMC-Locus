# Release 0.13.1 - Equipment UX Refoundation, Taxonomy And Configurable Field Templates

Version `0.13.1` is a focused productization patch for the Equipment
Repository.

## Added

- Equipment taxonomy with seven system root categories and default
  subcategories.
- Equipment field dictionary and category field rules.
- Effective category entry templates.
- Model creation from category template.
- Revision-aware custom field values and template snapshots.
- Searchable `root_category_id` and `is_demo` summaries.
- LAB CONSOLE Repository Administration and category-driven creation wizard.
- Demo hide/show/only filtering.

## Validation Scope

The release validates repository administration, template resolution, model
creation, audit/outbox evidence, and UI flow. It does not validate a complete
laboratory software product or any real acquisition chain.

## Limits

This release does not implement physical serial-numbered assets, station
binding, measurement-chain graphical drafting, live hardware drivers, real DAQ
runtime, FFT, report generation, RBAC/authentication, central sync, PostgreSQL,
Parquet, or HDF5.

The recommended next vertical is `0.14.0 - Physical Asset Fleet, Station
Connections And Measurement Chain Drafting`.
