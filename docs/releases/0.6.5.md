# EMC Locus 0.6.5 - Metrology Readiness and Audit/Outbox

## Delivered

Version `0.6.5` adds the first traceable readiness slice:

1. migration `0006_metrology_audit_events.sql` adds metrology audit events;
2. migrated metrology write paths require operation context;
3. instrument registration and calibration-event recording now write business
   data, metrology audit, and sync outbox records atomically;
4. serviceability changes are agent-backed and audit/outbox-backed;
5. readiness assessment returns structured ready/not-ready, per-instrument
   status, blocking issues, and warnings;
6. audit events can be queried for an instrument.
