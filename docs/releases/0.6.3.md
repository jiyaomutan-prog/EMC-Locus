# EMC Locus 0.6.3 - Agent-Backed Instrument Registry

## Delivered

Version `0.6.3` turns the 0.6.2 Rust metrology read boundary into the first
agent-backed instrument registry surface:

1. `storage init` now creates `metrology.sqlite` in addition to the project/sync
   databases;
2. `emc-locus-agent metrology register-instrument` registers a first instrument
   through Rust;
3. `emc-locus-agent metrology list-instruments` and `get-instrument` read the
   registry through the Rust service;
4. the loopback API exposes `GET/POST /api/v1/metrology/instruments` and
   `GET /api/v1/metrology/instruments/{asset_id}`;
5. `docs/metrology-api.md` documents the first metrology API contract and its
   limits.
