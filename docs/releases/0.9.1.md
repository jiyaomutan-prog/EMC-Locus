# EMC Locus 0.9.1 - Launchability And Template Hardening

Version `0.9.1` is a repair and hardening release on top of the 0.9.0
revisioned test-template aggregate. It does not introduce Template Studio, a
new GUI editor, a new execution runtime, real acquisition, FFT, or reporting.

## Added

- Windows launchers:
  - `scripts/start-proto.ps1` and `.bat`;
  - `scripts/start-qt-demo.ps1` and `.bat`;
  - `scripts/start-agent-qt.ps1` and `.bat`.
- Launcher logs under `logs/launchers`.
- `storage/sqlite/test_definitions/0005_single_active_draft.sql`, enforcing
  one active draft per template identity with a partial unique index.
- Python GUI-shell smoke test for strict bootstrap parsing and static paths.

## Changed

- `apps/gui-shell/bootstrap.js` is still loadable as a browser script, but the
  payload after `window.EMC_LOCUS_BOOTSTRAP =` is strict JSON for the Qt Python
  loader.
- Test-template aggregate DTOs expose:
  - `current_approved_revision`;
  - `latest_revision`;
  - `active_draft_revision`.
- Draft definition replacement uses SQL-level compare-and-swap on
  `definition_checksum`.
- Lifecycle transitions use SQL-level compare-and-swap on the current status.
- Approving a newer revision supersedes older approved revisions for the same
  template in the same transaction and emits audit/outbox evidence.
- Simulated EMC execution attempts preserve the selected approved template
  revision evidence when launched from a stored template id.

## Limits

- This is not a functional LAB CONSOLE template editor.
- This is not a full execution-package binding.
- This is not a real acquisition or post-processing runtime.
- PySide6 remains an optional local runtime dependency installed by launchers
  only when needed.

## Validation

Release validation is recorded in
`docs/session-logs/2026-07-01-release-0.9.1-session.md`. The final matrix
passed for Rust format, Clippy, Rust tests, Python compile/tests, SQLite
migration validation, release consistency, JavaScript syntax through the bundled
Node runtime, and Git whitespace checks.
